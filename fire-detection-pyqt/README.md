# 火灾识别检测系统

这是一个基于PyQt5和OpenCV的火灾识别检测系统。该系统可以通过摄像头实时监测火灾，或加载图像进行火灾检测。

## 功能特点
- 实时摄像头火灾检测
- 图像文件火灾检测
- 火灾检测状态显示
- 摄像头切换功能
- 简洁直观的用户界面

## 技术栈
- Python 3.8+
- PyQt5 (GUI框架)
- OpenCV (图像处理和火灾检测)
- NumPy (数值计算)

## 安装指南

### 1. 克隆或下载项目
将项目文件下载到本地目录。

### 2. 安装依赖
使用pip安装所需的依赖包：
```
pip install -r requirements.txt
```

### 3. 运行应用程序
执行main.py文件启动应用程序：
```
python main.py
```

## 使用说明

### 实时检测
1. 点击"开始检测"按钮启动摄像头实时检测
2. 系统会自动分析摄像头画面，检测是否有火灾
3. 检测状态会显示在界面上
4. 点击"停止检测"按钮停止实时检测

### 图像检测
1. 点击"加载图像"按钮选择要检测的图像文件
2. 系统会自动分析图像，检测是否有火灾
3. 如果检测到火灾，会弹出警告对话框

### 切换摄像头
点击"切换摄像头"按钮可以在多个摄像头之间切换

## 注意事项
- 确保您的计算机已安装Python 3.8或更高版本
- 确保您的摄像头工作正常
- 火灾检测算法基于颜色阈值，可能会受到光照条件等因素的影响
- 这是一个基础版本，实际应用中可能需要更复杂的算法和优化
