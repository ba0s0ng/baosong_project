# 工业互联网机床监测平台V1.0 使用说明

## 基本信息
- 软件名称：工业互联网机床监测平台V1.0
- 开发者：赵峰、王昊
- 开发单位：沈阳理工大学
- 版本：V1.0

## 快速启动
1. 双击 "启动平台.bat" 启动程序
2. 等待服务启动完成（约10-30秒）
3. 浏览器将自动打开并访问平台
4. 如浏览器未自动打开，请手动访问：http://127.0.0.1:8000

## 功能模块
1. **仪表盘** - 系统总览和实时监控
2. **机床管理** - 设备档案和状态管理
3. **数字孪生** - 三维可视化和仿真
4. **数据库管理** - 数据查询和维护
5. **报警管理** - 异常监测和报警处理
6. **数据分析** - 性能分析和趋势预测

## 系统要求
- 操作系统：Windows 10/11
- 内存：最低2GB，推荐4GB以上
- 硬盘：最低500MB可用空间
- 浏览器：Chrome、Firefox、Edge等现代浏览器

## 注意事项
1. 首次启动可能需要较长时间，请耐心等待
2. 请确保8000端口未被其他程序占用
3. 如遇到问题，请查看控制台输出信息
4. 关闭程序请直接关闭命令行窗口
