# 智能语音问答系统

## 项目概述
这是一个基于PyQt5的本地AI聊天客户端，集成了Ollama本地API，支持与DeepSeek等本地模型交互。系统提供文本对话、语音播报等功能，并具有美观的用户界面。

## 功能特点
1. 基于PyQt5的图形用户界面，提供美观的聊天体验
2. 集成Ollama本地API，支持与DeepSeek等本地模型交互
3. 支持文本对话功能，带有用户和AI的消息气泡显示
4. 可选文本转语音(TTS)功能，支持语音播报AI回复
5. 自适应界面布局，消息气泡根据内容自动调整大小
6. 错误处理和状态提示机制
7. 完全本地运行，保护隐私
8. 响应式设计，适应不同窗口大小
9. 跨平台支持(Windows/macOS/Linux)
10. 可配置的模型选择和API设置

## 安装说明
1. 确保已安装Python 3.7或更高版本
2. 安装必要依赖:
   ```
   pip install -r requirements.txt
   ```
3. 确保Ollama服务已启动并运行在localhost:11434

## 使用方法
1. 运行主程序:
   ```
   python dsk-tts.py
   ```
2. 在界面中输入文本消息并发送
3. 可选择启用TTS功能，聆听AI回复
4. 使用音量滑块调整TTS音量

## 依赖库
- PyQt5: 图形界面
- requests: HTTP请求
- pyttsx3: 文本转语音

## 注意事项
- 确保Ollama服务正在运行
- 首次使用前请安装所有依赖库
- 对于语音功能，可能需要根据操作系统配置音频设备
