# 工业互联网机床监测平台

## 项目概述

工业互联网机床监测平台是一个基于工业物联网技术的机床设备状态监测与管理系统。该平台旨在帮助制造企业实时监控机床运行状态，收集和分析设备数据，预测潜在故障，提高生产效率和设备利用率。

## 功能特点

### 核心功能
- **设备状态实时监控**：实时显示机床运行参数和状态信息
- **数据统计与分析**：提供设备运行数据的统计分析、图表生成和趋势预测
- **异常报警管理**：负责机床运行异常报警的接收、显示、处理和记录
- **历史数据查询**：支持查询和导出设备历史运行数据
- **设备信息管理**：负责机床设备的信息展示、状态监控和参数配置

### 技术特点
- **前后端分离架构**：采用现代化前后端分离架构，提供良好的用户体验
- **实时数据通信**：通过MQTT协议实现设备数据的实时传输
- **轻量级数据存储**：使用SQLite数据库进行数据存储，部署简单便捷
- **响应式界面设计**：前端界面采用响应式设计，支持多种设备访问

## 系统架构

### 前端架构
- **框架技术**：HTML5, CSS3, JavaScript
- **UI框架**：Bootstrap
- **图表展示**：Chart.js或其他图表库
- **通信方式**：AJAX调用后端API

### 后端架构
- **Web框架**：FastAPI (Python)
- **服务器**：Uvicorn
- **数据存储**：SQLite
- **消息通信**：MQTT (paho-mqtt)
- **数据模型**：Pydantic

## 安装部署

### 环境要求
- **Python**：3.8+
- **pip**：最新版
- **浏览器**：Chrome 90+, Firefox 88+, Edge 90+

### 后端部署

1. 进入后端目录
```bash
cd backend
```

2. 安装依赖包
```bash
pip install -r requirements.txt
```

3. 启动后端服务
```bash
python -m app.main
```

后端服务将在 http://0.0.0.0:8000 上运行

### 前端部署

1. 进入前端目录
```bash
cd frontend
```

2. 启动HTTP服务器（示例使用Python的内置服务器）
```bash
python -m http.server 8080
```

前端服务将在 http://localhost:8080 上运行

## 使用说明

### 访问系统
在浏览器中访问 http://localhost:8080 进入系统登录页面

### 主要模块使用
1. **设备监控模块**：查看所有机床设备的实时运行状态和参数
2. **数据分析模块**：查看设备历史运行数据的统计图表和分析报告
3. **报警管理模块**：查看和处理设备异常报警信息
4. **设备管理模块**：管理机床设备的基本信息和参数配置

## 故障排除

### 常见问题
1. **后端服务启动失败**
   - 检查端口8000是否被占用
   - 确认依赖包是否安装完整

2. **前端无法连接后端**
   - 检查后端服务是否正常运行
   - 确认API地址配置是否正确

3. **数据库相关问题**
   - 确认data目录下的SQLite数据库文件权限
   - 必要时可删除数据库文件，系统将自动重建
